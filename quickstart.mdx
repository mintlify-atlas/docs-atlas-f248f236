Get your music player running locally and import your first tracks.

## Prerequisites

<CardGroup cols={2}>
  <Card title="Node.js 20+" icon="node-js">
    Download from [nodejs.org](https://nodejs.org/)
  </Card>
  <Card title="pnpm" icon="box">
    Install with `npm install -g pnpm`
  </Card>
</CardGroup>

## Quick Setup

<Steps>
  <Step title="Clone the repository">
    ```bash
    git clone https://github.com/cdxker/rotations.git
    cd rotations
    ```
  </Step>

  <Step title="Install dependencies">
    ```bash
    pnpm install
    ```
    
    This installs all packages in the monorepo.
  </Step>

  <Step title="Start the music player">
    ```bash
    cd site
    pnpm dev
    ```
    
    The player will open at [http://localhost:4321](http://localhost:4321)
  </Step>

  <Step title="Import your first music">
    1. Click **Add Music** button
    2. Choose **Bandcamp** as your source
    3. Paste a Bandcamp album URL
    4. Click **Import**
    
    <Note>
      Example URL: `https://bandcamp.com/album/your-favorite-album`
    </Note>
  </Step>

  <Step title="Start listening">
    - Click any track to play
    - Track automatically added to your Current Rotation
    - Audio cached for offline playback after first stream
  </Step>
</Steps>

## Optional: Graph Visualization

Want to see your listening patterns as a graph? Set up the graph server:

<Steps>
  <Step title="Start Postgres">
    ```bash
    docker compose up -d
    ```
  </Step>

  <Step title="Configure environment">
    ```bash
    cd graph-server
    cp .env.example .env
    ```
    
    Edit `.env` and add your Last.fm API key:
    ```env
    LASTFM_API_KEY=your_api_key
    DATABASE_URL=postgresql://postgres:postgres@localhost:5432/graph
    ```
    
    <Info>
      Get a Last.fm API key at [last.fm/api/account/create](https://www.last.fm/api/account/create)
    </Info>
  </Step>

  <Step title="Start graph server">
    ```bash
    cd graph-server
    pnpm dev
    ```
  </Step>

  <Step title="Import your scrobbles">
    ```bash
    curl -X POST http://localhost:3001/pipeline/run \
      -H "Content-Type: application/json" \
      -d '{"username":"your_lastfm_username"}'
    ```
  </Step>

  <Step title="Open graph visualization">
    ```bash
    cd graph-frontend
    pnpm dev
    ```
    
    Visit [http://localhost:3000](http://localhost:3000)
  </Step>
</Steps>

## Troubleshooting

<AccordionGroup>
  <Accordion title="Port already in use">
    If port 4321 is taken, edit `site/astro.config.mjs` to change the port.
  </Accordion>

  <Accordion title="pnpm install fails">
    Make sure you're using Node.js 20 or higher:
    ```bash
    node --version
    ```
  </Accordion>

  <Accordion title="Bandcamp import not working">
    Some Bandcamp albums require authentication. Try a different public album first.
  </Accordion>

  <Accordion title="Docker not starting">
    Make sure Docker Desktop is running and you have permission to use Docker.
  </Accordion>
</AccordionGroup>
